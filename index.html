Mobile Hour Website

As a web developer, I undertook a study project to completely revamp The Mobile Hour's website. Using PHP and MySQL, I created a responsive and secure online store that exceeded the client's expectations. The website showcased a modern design and enhanced functionality, allowing customers to browse and filter mobile phones by brand, price, and stock availability. Admin users had secure access to manage products, users, and track changes with a comprehensive change log.

The T-Shirt Grind Online shop

In this study project, I designed and developed an online shop website for 'The T-Shirt Grind' using the WordPress CMS platform. The website catered to the client's specific needs by offering a specialized shopping cart for their best-selling custom t-shirts, categorized by genre. Customers could easily select size, style, and color options for each design, while stock management ensured accurate inventory control. Integration with PayPal provided secure payment transactions, and a user-friendly interface allowed for order tracking.
