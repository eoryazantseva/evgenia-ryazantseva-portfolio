Summer Escapes Project

Summer Escapes stands as my maiden project during my studies at TAFE Queensland. This responsive website was crafted for 'Summer Escapes,' a budding startup that focuses on providing insights into local summer holiday destinations. The task revolved around addressing mobile device compatibility issues, culminating in the creation of four webpages: Home, Reviews, Feedback (complete with a user-friendly form), and Contact. All these pages feature a shared external CSS style sheet, ensuring a uniform look and feel. The website dynamically adapts to various screen sizes, allocating 80% of the screen for desktop, 100% for tablets, and fully optimizing phone layouts. Summer Escapes sets the stage for a comprehensive and intuitive travel experience, no matter the device.
